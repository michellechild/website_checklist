HTML

  • Did you include at the top of the html document?
  • Did you include a language specification?
  • Did you specify a meta tag?
  • Is your CSS reset file before the CSS file?
  • Also, do you have a CSS reset or normalize file? You should.
  • Are br tags used for type purposes only?
  • Did you include alt tags for all images?
  • Is the h1 tag used only once?
  • Is there a containing div? If not, GET ONE.
  • Are all characters coded?
  • Does anything reside between ul tags and li tags? If so, GET RID OF IT.
  • Are any id tags used only once?
  • Are all class and id tags labeled based on content, not style?
  • Are all assets referenced correctly?
  • Is the html document indented and spaced correctly?

  • Are all elements, classes and ids categorized correctly?

  • Are all brackets closed?
  • Is the CSS document indented and spaced correctly in a readable manner?

Technical Specifications and File Management

  • Is the first file named index.html?
  • Are style sheets in a separate folder?
  • Are assets in a separate folder?
    • Are assets labeled clearly and based on content?
  • Are all versions clearly labeled?
  • Have you deleted any unnecessary files?

Content

  • Is the content yours?
    • If not, have you clearly credited the author?
  • Is the content clearly written and easy to understand?
  • Is the content the focus of the site?

Design Foundations

Grid

  • Is this site (mostly) aligned to a grid?
  • Is there a sufficient amount of whitespace, created with margins and padding?
  • Were Gestalt Theory Principles considered with the overall design?

Typography

  • Do the fonts work well together?
    • Are you sure?
    • Do the x-heights match?
    • Do they set the tone for the content?
  • Do you have these five properties styled for each typographic element in your CSS?
    1. Font-family, including second, third and fourth options
    2. Font-size (at least 16px for web)
    3. Line-height (140% minimum, but ultimately based on font-size.)
    4. Color
    5. Max-Width: 45 to 90 characters per line

Color Theory

  • Do the colors reflect your message?
  • Test the color contrast of your type and background colors
  • Test for color blindess compatibility

Saving Images

  • Follow this order when saving images for the web:
    1. Make sure all images are sized proportionally.
    2. Crop images proportionally and carefully to fit size specifications.
    3. Choose the best filetype for your image
    4. Use the Save For Web feature on Adobe Illustrator or Photoshop. Here you can alter the quality, size and preset. Choose the correct filetype with the lowest quality setting that will still produce a clear image on the web.
