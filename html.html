HTML

  • Your document starts like this: <!DOCTYPE html>
    <html lang="en-US">
    <head>
    <title>Web Design is Cool</title>
  • Links to your CSS stylesheets. <link href="stylesheets/style.css" type="text/css" rel="stylesheet">
  • Includes a meta tag for charset to declare the character encoding used of the page. <meta charset="utf-8">
  • Includes a meta tag for viewport to allow devices to read media queries (see responsive web design). <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • Links to web fonts from services like Adobe TypeKit, myFonts, Webtype, etc. (Adobe Typekit example here). <script src="//use.typekit.net/fxv0szo.js"></script> <script>try{Typekit.load();}catch(e){}</script>
  • And finally, links to your .js file or jQuery hosting site (this can also be placed at the bottom of your body). <scriptsrc="//code.jquery.com/jquery-1.11.0.min.js"></script><head>
  • <br> tags are used for type purposes only.
  • <alt> tags are included on all images.
  • <h1> tag is only used once per page.
  • **Container div is wrapped around content!**
  • Characters are coded: & ‘ ’ “, ”
  • Nothing resides between <ul> and <li>tags.
  • <id> tags are used only once.
  • HTML document is indented and spaced correctly. If you're using Sublime Text, try package HTML Beautify.
